### About App

- <B>Language:</B> Java
- <B>Libraries:</B>
  - Android Jetpack(ViewModel, LiveData, DataBinding, Room, Navigation)
  - Retrofit, RxJava
- <B>Features:</B>
  - 안드로이드 권장 앱 아키텍처, 클린 아키텍처 기반으로 앱 구조 설계
  - Retrofit, RxJava를 통한 비동기 API 통신
  - Room을 통한 네트워크 캐시
