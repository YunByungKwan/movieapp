### About App

- <B>Language:</B> Java
- <B>Libraries:</B>
  - Android Jetpack(ViewModel, LiveData, DataBinding, Room, Navigation)
  - Retrofit, RxJava
- <B>Features:</B>
  - Android Jetpack을 사용하여 MVVM 패턴으로 설계
  - Retrofit, RxJava를 통한 비동기 API 통신
  - Room을 통한 네트워크 캐시
